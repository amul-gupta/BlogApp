# 📝 BlogApp - Spring Boot Blogging Application

A full-stack backend **Blog Application** built using Spring Boot.  
It provides REST APIs for managing users, posts, categories, and comments with secure authentication using **JWT (JSON Web Token)**.

---

## 🚀 Features

- 👤 User registration and login (JWT Authentication)
- 📝 Create, update, delete blog posts
- 📂 Category management for posts
- 💬 Comment system on posts
- 🔐 Role-based access control (User/Admin)
- 🛡️ Secure REST APIs using Spring Security
- 🗄️ MySQL database integration using JPA/Hibernate
- ⚠️ Global exception handling

---

## 🛠️ Tech Stack

- Java
- Spring Boot
- Spring Security
- Spring Data JPA
- Hibernate
- MySQL
- JWT (JSON Web Token)
- Maven
